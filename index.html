CSS INTERVIEW QUESTIONS DAY 1


1. What is box sizing.

The CSS box-sizing property allows us to include the padding and border in an element's total width and height.



2. What is animation delay.

The animation-delay property specifies a delay for the start of an animation. The animation-delay value is defined in seconds (s) or milliseconds (ms).



3. Which property is used to merge cells vertically in grid

The grid-column property specifies a grid item's size and location in a grid layout



4. Which property is used to merge cells horizontal in grid

The grid-row property



5. What is CSS box model.

The CSS box model is a container that contains multiple properties including borders, margin, padding, and the content itself. It is used to create the design and layout of web pages



6. Difference between display none and visibility hidden.

display none = It removes the element from the document. It does not take up any space.

visibility hidden = It hides the element, but it still takes up space in the layout.
